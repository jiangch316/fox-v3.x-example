# Fox

Fox Router and commons Utils

## 安装

正式版

```js
npm install @fox-js/fox@next
```

测试版

```js
npm install @fox-js/fox@beta
```

## 使用

```js
import fox from '@fox-js/fox';
```

## 编译

开发版本

```js
npm run dev
```

正式版本

```js
npm run prod
```

## 发布

测试版本

```js
npm run publish:beta
```

正式版本（慎重）

```js
npm run publish:next
```
